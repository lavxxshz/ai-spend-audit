# Metrics

## North Star Metric
**Audits completed per week**

Why: An audit completed means a user got value. Everything else — email captures, Credex consultations, shares — flows from audit completion. If audits go up, everything goes up. If audits plateau, no amount of funnel optimization saves us.

## 3 Input Metrics That Drive the North Star

1. **Landing page → audit start conversion rate** (target: 60%)
   If this drops, the landing page copy or load time is the problem.

2. **Audit start → audit completion rate** (target: 80%)
   If this drops, the form is too long or confusing. Simplify.

3. **Weekly new traffic sources** (target: 3+ distinct channels)
   Diversification prevents single-channel dependency. Track Reddit, HN, Twitter, direct separately.

## What to Instrument First

1. Audit completion event with tool count and total spend
2. Email capture conversion rate
3. Share URL clicks (how many people view shared audits)
4. Credex banner click-through rate
5. Time from landing to audit completion

## Pivot Trigger

If after 30 days:
- Audit completion rate < 40% → form is too complicated, simplify to 3 fields
- Email capture rate < 10% → value proposition not clear enough, redesign results page
- Credex click rate < 5% → Credex placement needs to be more prominent or better copy
- Total audits < 100 → distribution problem, double down on Reddit/HN outreach
